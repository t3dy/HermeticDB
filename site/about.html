About This Project

The EmeraldTablet Knowledge Portal is a digital humanities project providing encyclopedic reference for the Hermetic tradition from late antiquity to the early modern period, anchored on a digital edition of the Emerald Tablet (Tabula Smaragdina).

Architecture

SQLite database → Python pipeline → static HTML/CSS/JS → GitHub Pages. No frameworks, Python standard library only.

Provenance

Every datum in this portal carries provenance metadata: source method (SEED_DATA, CORPUS_EXTRACTION, DETERMINISTIC), review status (DRAFT, REVIEWED, VERIFIED), and confidence level (HIGH, MEDIUM, LOW).

Methodology

Data pipeline: corpus conversion (PDF to markdown) → corpus indexing → text segmentation → keyword marking → entity extraction → relationship linking → card assembly → static site generation → validation.
